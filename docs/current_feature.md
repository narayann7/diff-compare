# DiffCompare Features

Here is a list of features available in the DiffCompare app:

- **Multiple Diff Views**: Choose between Split (Side-by-side) and Unified view modes.
- **File Upload & Editing**: Load text or code files directly or write/paste content into the Original and Modified editor panels.
- **Diff Statistics**: View a clear summary of added, removed, and unchanged lines.
- **Ignore Whitespace**: A toggle to ignore or include whitespace changes in your diff comparisons.
- **Minimap Support**: Navigate through long files easily using the minimap.
- **Expand/Collapse View**: A distraction-free, expanded view of the diff output with keyboard shortcut support (`Cmd/Ctrl + E`).
- **Copy Functionality**: Easily copy original/modified text or copy the entire unified/split diff output with a single click.
- **Multiple Color Themes**: Choose from various beautiful themes including Dark, Light, Dracula, Ocean, and Skillz.
- **Reset Functionality**: Quickly clear all inputs or start over with a single click.
- **Animated Diff Replay**: Render an animated replay of your diffs using the fully integrated Remotion animation modal.
